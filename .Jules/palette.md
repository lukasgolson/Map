## 2025-01-01 - Add Focus Visible Styles for Accessibility
**Learning:** Found that neon buttons in Dino-Transit lack focus indicators for keyboard navigation, making it hard for screen reader or keyboard-only users to know which element is currently focused.
**Action:** Adding `:focus-visible` styles with a neon glow to elements like `.neon-btn` to improve keyboard navigation accessibility while maintaining the retro neon aesthetic.
